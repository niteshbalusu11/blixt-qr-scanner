# blixt-qr-scanner

Scans QR Codes without google mlkit

## Installation

```sh
npm install blixt-qr-scanner
```

## Usage

```js
import { multiply } from 'blixt-qr-scanner';

// ...

const result = await multiply(3, 7);
```
